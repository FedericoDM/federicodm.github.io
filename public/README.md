# Public Assets

This folder contains static assets that will be publicly accessible.

## Resume

To add your resume:
1. Export your resume as a PDF file
2. Name it `resume.pdf`
3. Place it in this `public/` folder
4. The resume will be accessible at `/resume.pdf` on your website

The "Download Resume" link in the Contact section will automatically link to this file.

## Other Assets

You can also add other files here:
- Images (logos, photos, etc.)
- Documents
- Favicons
- Any other static files

All files in this folder will be accessible from the root URL of your site.
